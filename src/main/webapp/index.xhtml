Descrição problema

Você recebeu uma demanda no seu trabalho para desenvolvimento de um projeto Java para web. Para isso, acompanhe as seguintes especificações:

Uma academia deve manter informações sobre seus clientes. Inicialmente, será necessário armazenar, para cada cliente, as seguintes informações: nome, matrícula, endereço completo, bem como em qual modalidade esportiva está matriculado. Cabe ressaltar que um cliente pode estar vinculado a apenas uma modalidade, embora cada modalidade possa ter vários atletas. Cada atividade da academia pertence a uma modalidade. Assim, a academia deseja manter um cadastro para todas as modalidades existentes, com as seguintes informações: código da modalidade e descrição.

Com base na descrição do cenário acima, você deverá desenvolver uma aplicação utilizando o modelo MVC (Model-View-Controller) com Servlet e JSP. Esta aplicação deverá inserir, editar, listar e excluir registros em um banco de dados do SGBD MySQL Server, conforme as especificações apresentadas.
